About Me

I'm Shuyu Zheng from Xiamen, Fujian, China.

I'm now a second-year CS Master student at Peking University, China, advised by Prof. Xuanzhe Liu. Previously, I received my CS bachelor degree in Peking University.

My research focus on measuring, analysing and modeling "real-world" large scale user behavior/interactions. My current projects also cover the cutting-edge topics such as on-device machine learning and WebAssembly.

In my spare time, I have a wide interest in literature, music, fashion design, handwork... And I'm always looking for something new!

Projects

Service Access Pattern Analysis

This work is in cooperation with Alibaba Group. We conduct an empirical study on service access patterns via heterogeneous service clients based on two large-scale real-world datasets of launch events. We explore how users consume online services through different service clients, and to what extent their usage behavior patterns vary when consuming the same service.

Performance Measurement of On-browser Deep Learning Frameworks

We conduct the first empirical study of deep learning in browsers, surveying 7 most popular JavaScript-based deep learning frameworks and digging out the performance gap between deep learning in browsers and on native platforms. Cooperate in a team of four advised by Professor Xuanzhe Liu, co-director of SaaS Research Group(System Research Lab, Institute of Software).

Web GUI Optimization based on Design Mining

Inspired by typical design process of human designers, we propose a data-driven automation system named FaceOff that assists the manifestation design of web GUI. We address the task by jointly training structure embedding and style discriminating networks. The first network supports structural retrieval, and the second aims to preserve overall visual harmonization. [Demo]

Game: Splender

An online version of Splender, a tabletop game of three. I redesigned the user interface of the game, providing a simplified, interative and hearting-warming user interface. Players can battle with AIs or play online with friends.

Portrait MaSter

We build Portrait MaSter, a tool that extracts portrait from an image and apply a style transfer according to user preferences. The system contains a demo frontend, a backend server and the inference pipeline. The overall inference process includes the human portrait matting stage and the style transfer stage.

Evol Calculator

A Web App that solves puzzles in the mobile game Evol which calculates the best card strategy. The calculator contains a responsive front end, a server and a database. [Open Source Code]

JG Helper (Wechat Mini Program)

A Wechat mini program for students to do quizzes and a webpage for teachers to edit tests, view results and details for the course Introduction to Computation. The project can be found on the Wechat mini program platform.

Takeaway Helper (Android App)

We Designed and developed an Android App that rearranges restaurant information from three major takeaway platforms. This App provides recommendation based on user behavior and real time discount information.
